Getting Started

Opening a Markdown File

There are several ways to open a Markdown file in MarkDown Reader:

  • Double-click any .md file in Finder (if MarkDown Reader is set as your default viewer).
  • Choose File › Open (or press ⌘O) and select a Markdown file.
  • Drag and drop a .md file onto the MarkDown Reader icon in the Dock.
  • Choose a file from File › Open Recent.

Set as Default Markdown Viewer

To open all Markdown files with MarkDown Reader by default:

  1. Right-click any .md file in Finder.
  2. Select Get Info (or press ⌘I).
  3. In the "Open with:" section, choose MarkDown Reader from the dropdown.
  4. Click Change All... to apply to all .md files.
  5. Confirm when prompted.
Tip: MarkDown Reader supports files with extensions .md, .markdown, .mdown, .mkd, and .mkdn.

The Interface

When you open a document, you'll see:

  • Toolbar — toggle the table of contents, zoom controls, search, print, and export buttons.
  • Table of Contents (sidebar) — shows all headings in the document for quick navigation. Click any heading to jump to it.
  • Document area — the rendered Markdown content.
